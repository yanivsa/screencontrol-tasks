# Privacy Policy

ScreenControl Tasks is a private family task and screen-time wallet app.

## Data Collected

The app stores:

- Parent and child profile names.
- Child task assignments and task completion status.
- Screen-time minute balances and transaction history.
- Optional task proof photos uploaded by a child.
- Internal app notifications.

## Children Data

The app is intended for private use by one family. Child data is used only to operate the task and screen-time minute system. The app does not sell data, share data with advertisers, or use data for behavioral advertising.

## Photos

Photos are used only as task proof for parent review. Photos are deleted according to the app retention policy, currently after 7 days where cleanup is available.

## Authentication

The app uses PIN-based family login. PINs are stored as hashes, not plain text. Sessions use signed tokens.

## Third Parties

The app uses Cloudflare for hosting, API execution, database storage, and optional object storage.

## Contact

For privacy questions, contact the app owner through the Google Play listing account.
